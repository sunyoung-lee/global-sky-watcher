## 요약
Global Sky Watcher는 3D 인터랙티브 지구본 위에 전 세계의 실시간 비행 데이터를 시각화하는 항공 관제 시뮬레이션 대시보드입니다. 이 제품은 사용자에게 시각적 경이로움과 실시간 연결감을 제공하며, 방대한 지리 정보를 직관적으로 소비할 수 있는 새로운 방식을 제시합니다.

## 타겟 시장
- **핵심 사용자 페르소나**: 기술에 익숙하며, 실시간 정보 소비를 즐기는 30-40대 남성, 항공 산업에 종사하거나 관심이 높은 사용자.
- **시장 기회**: 글로벌 항공 애호가, 여행자, 항공 산업 관계자를 대상으로 하며, 특히 실시간 데이터 소비와 시각적 경험을 중시하는 사용자를 타겟으로 합니다.

## 핵심 기능 (MVP)
1. **Real-time 3D Globe**: 고퀄리티 3D 지구본을 제공하여 사용자가 전 세계를 탐색할 수 있습니다.
2. **Live Flight Mapping**: 실시간 비행기 위치를 매핑하고, 고도에 따라 색상을 달리하여 입체감을 부여합니다.
3. **Flight Detail Info**: 비행기 클릭 시 상세 정보를 제공하는 인포메이션 카드 출력.
4. **Jeju Auto-Focus**: 앱 실행 시 제주도 상공을 비추며 시작하는 기능.

## 기술 아키텍처
- **프론트엔드**: `React 19`, `Three.js`, `globe.gl`(vanilla)을 사용하여 3D 그래픽과 UI를 구현.
- **백엔드**: `Node.js`와 `Express`를 사용, WebSocket을 통해 실시간 데이터 푸시.
- **Edge 프록시**: Vercel Edge Runtime으로 OpenSky API 프록시 (CORS 우회 + 서버 인증).
- **고수준 시스템 설계**: 클라이언트가 OpenSky를 직접 호출(1차), 실패 시 Edge 프록시 폴백(2차).

## 디자인 방향
- **UX 원칙**: 직관적인 인터랙션과 즉각적인 피드백 제공.
- **비주얼 방향**: 밤하늘 테마로 어두운 배경과 노란색, 청록색의 비행기 아이콘을 사용하여 시각적 집중 유도.

## 품질 및 테스트 전략

- **품질 기준**: 안정성, 실시간성, 사용자 경험을 보장.
