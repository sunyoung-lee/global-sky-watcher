# Global Sky Watcher

3D 인터랙티브 지구본 위에 전 세계 실시간 비행 데이터를 시각화하는 항공 관제 시뮬레이션 대시보드입니다.

## 핵심 기능

- **Real-time 3D Globe** - 고퀄리티 3D 지구본으로 전 세계 탐색
- **Live Flight Mapping** - 실시간 비행기 위치 매핑 (고도별 색상 구분)
- **Flight Detail Info** - 비행기 클릭 시 상세 정보 카드 출력
- **Jeju Auto-Focus** - 앱 실행 시 제주도 상공에서 시작

## 기술 스택

- **Frontend**: React, Three.js, react-globe.gl
- **Backend**: Node.js, Express, WebSocket
- **Data**: OpenSky Network API
